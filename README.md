# Rails Manual CRUD 
    Rails v5.1.5

    PostgreSQL

    Bootstrap 3.0

# Cara Install

    $ git clone https://github.com/maulayyacyber/rails-manual-crud.git

    $ cd rails-manual-crud

    $ bundle install

    $ rake db:create

    $ rake db:migrate

    $ rails s
